# เข็มทิศอาชีพ RIASEC Thailand

ต้นแบบเว็บแบบไม่ต้อง deploy สำหรับให้นักเรียนทำแบบประเมิน RIASEC บนมือถือหรือคอมพิวเตอร์ และดูข้อมูลอาชีพ/การศึกษาต่อในประเทศไทย

## การใช้งาน

เปิดไฟล์ `index.html` ด้วยเว็บเบราว์เซอร์ได้ทันที (ดับเบิลคลิกไฟล์) โดยไม่ต้องติดตั้งโปรแกรมหรือเชื่อมต่อเซิร์ฟเวอร์

- แบบประเมิน 36 ข้อ แบ่งเป็น กิจกรรม ความสามารถ และอาชีพ
- คำนวณ Holland Code 3 ตัวอักษรอัตโนมัติ
- ส่งออกผลลัพธ์เป็น CSV ซึ่งเปิดด้วย Microsoft Excel ได้
- ฐานอาชีพ 30+ อาชีพ พร้อมคณะ/สาขาที่เกี่ยวข้อง
- ลิงก์ MyTCAS, กระทรวงแรงงาน, กรมการจัดหางาน และเว็บไซต์สมัครงานของไทย

## หากนำไปใช้กับ Glide + Google Sheets

นำไฟล์ CSV ที่ส่งออกไปเก็บใน Google Sheets ได้โดยตรง แล้วใช้คอลัมน์ `Holland Code` และคะแนน R/I/A/S/E/C สร้างหน้าผลลัพธ์ใน Glide ได้ ระบบนี้ไม่ส่งข้อมูลไปภายนอกเอง เพื่อคงความเป็นส่วนตัวของนักเรียน

> ข้อควรทราบ: ผล RIASEC เป็นเครื่องมือสำรวจความสนใจ ไม่ใช่การตัดสินความสามารถหรืออนาคตของนักเรียน
